# music

这是一个根据网易云音乐api 实现的一个音乐播放器
 网易云api 地址：https://github.com/Binaryify/NeteaseCloudMusicApi.git
 
 ## 项目运行
``` bash
# 克隆到本地
git clone 

# 进入文件夹
cd vuemusic

# 安装依赖
npm install(推荐)

# 开启本地服务器
npm run serve

# 发布环境
npm run build
```
